# Introduction
This is including all commands related to Server for a discord server.

## Commands
* embed - Create an embed
* poll - Create a poll
* reactrole - Create a role-reaction message (up to 5 reactions/roles in one embed)
* configuration - IMPORTANT] Configure the bot for the server (see configuration.mdx)

### `/help`
The `/help` command provides a list of available commands and their descriptions.
