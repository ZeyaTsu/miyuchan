# Introduction
This is including all commands related to moderation for a discord server.

## Commands
* ban - Ban a user
* kick - Kick a user
* warn - Warn a user
* unwarn - Remove a warning from a user
* clear-warn - Remove all warnings from a user
* clear - Delete a certain amount of messages 
* banwords - Add or Remove words in the banwords list, when a member says one, the message is directly deleted
* banwords_list - See the  banwords list

## Ctx commands
* Report - Right click on a message > Applications > Report

### `/help`
The `/help` command provides a list of available commands and their descriptions.
