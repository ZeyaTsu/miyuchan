# Introduction
This is including all information related to Configuration for a discord server.

## Why is configuration important
Configuration is important for the bot if you want to use it with no issues. It is including the channels for Report, Level, Welcome but also features such as Auto-nick/Auto-role when a member joins the server.
Depending on your goals you may not want to miss it
If one is set to 'None' (default) there won't be any messages.

The `/help` command provides a list of available commands and their descriptions.
