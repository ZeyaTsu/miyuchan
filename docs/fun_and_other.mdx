# Introduction
This is including all commands related to Fun & Other for a discord server.

## Commands
* hug -  Hug someone
* Punch - Punch someone
* lovecalc - See how much is the love between you and someone else
* ping - Know how is your connection going
* news - Check latest news & updates related to Miyuchan
